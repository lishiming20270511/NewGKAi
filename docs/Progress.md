## 9:16 直播竖屏 UI 设计（2026-06-24）

### 背景
主播在 PC 端用 OBS 做竖屏直播，手机端观众反馈现有页面（横屏 1280px）在直播窗口效果差。本次设计针对 9:16 竖屏直播场景制作登录页原型。

### 原型迭代

| 版本 | 文件 | 关键变化 |
|------|------|----------|
| v3 | `docs/prototype_9x16_login_v3.html` | 暗色基础框架；`width:100vw`；flex 20/30/40/10 比例 |
| v4 | `docs/prototype_9x16_login_v4.html` | 亮色主题（`#F0F4FF` + 白卡片 + 蓝渐变按钮） |
| v5 | `docs/prototype_9x16_login_v5.html` | SimHei/黑体优先；全局 +5px；全黑文字 |
| v6 | `docs/prototype_9x16_login_v6.html` | 全局再 +5px |
| v7 | `docs/prototype_9x16_login_v7.html` | 全局再 +10px |
| **v8** | **`docs/prototype_9x16_login_v8.html`** | **直播清晰度优化**：关闭抗锯齿 + text-shadow 笔画加粗 + 纯黑 `#000000` + 边框加粗至 2px |

### 最终规格（v8）

- 布局：`width:100vw`，无 max-width，flex column，品牌/登录/功能/页脚 = 20/30/40/10
- 字体：SimHei > Heiti SC > STHeiti > 黑体 > Microsoft YaHei
- 字号：品牌名 46~60px，输入框 38~44px，登录按钮 42~50px，功能标题 37~43px
- 配色：背景 `#F0F4FF`，卡片 `#FFFFFF`，文字 `#000000`，强调 `#2563EB`

### 最终决策（已执行）

- [x] v8 风格已应用到生产 `frontend/index.html`（全站：SimHei/黑体、#F0F4FF 背景、纯黑文字、大字号、抗锯齿关闭）
